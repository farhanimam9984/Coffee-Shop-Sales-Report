* Project Overview

This project focuses on analyzing Coffee Shop Sales Data to extract meaningful business insights related to sales performance, customer behavior, product demand, store performance, and time-based trends.
I used Microsoft Excel as the primary tool, leveraging Power Query, Pivot Tables, Pivot Charts, and an interactive dashboard to convert raw, messy data into a clean, structured, and decision-ready report.
The final output is a fully interactive dashboard that helps stakeholders quickly understand sales patterns and make data-driven decisions.

-> Tools & Techniques Used

Microsoft Excel
Power Query (for data cleaning & transformation)
Pivot Tables & Pivot Charts
Excel Dashboard Design
Basic Trend Prediction & Time-Based Analysis


Steps By Step Procedure:

1.Understanding the Raw Data
The dataset contained coffee shop transaction-level data, including:
Transaction ID
Product category & product name
Store location
Order size
Date, hour, and weekday
Sales amount

-> Challenge:
The data was messy and unstructured, with inconsistent formats, unnecessary columns, and values that were not analysis-ready.

2.Data Cleaning & Transformation using Power Query
I used Power Query because it is:
Efficient for large datasets
Repeatable and scalable
Industry-relevant for Excel-based data analysis
Key cleaning steps performed:
Removed duplicate and irrelevant records
Standardized column names and formats
Converted date and time fields into usable formats
Created new calculated columns:
Hour of order
Weekday
Month
Handled missing and undefined values properly

-> Why Power Query?
As a Data Analyst, using Power Query ensures:
Automation of cleaning steps
Consistency when data updates
Cleaner pivot table creation downstream

3.Creating Pivot Tables for Analysis
After cleaning the data, I created multiple Pivot Tables, each answering a specific business question:

🔹 Sales Performance
    Total Sales Revenue
    Average Order Amount
    Total Footfall (number of transactions)

🔹 Time-Based Analysis
    Sales by Hour (peak ordering time)
    Sales by Weekday
    Monthly sales trends and prediction

🔹 Product Analysis
    Sales by Product Category (Coffee, Tea, Bakery, etc.)
    Top 5 products based on sales
    Product size distribution (Small, Regular, Large)

🔹 Location-Based Analysis
    Store-wise sales comparison
    Footfall by store location

-> Why Pivot Tables?
Pivot tables allow:
Fast summarization of large datasets
Dynamic filtering
Easy integration with charts and dashboards

4.Dashboard Design & Visualization
Using Pivot Charts and Excel formatting, I designed an interactive dashboard that includes:
🔹KPI Cards:
   Total Sales
   Footfall
   Average Order Value
🔹Visual Insights:
     Line chart: Orders by hour
     Pie charts: Category-wise and size-wise distribution
     Bar charts: Top products and store-wise sales
     Weekday order analysis
🔹Slicers & Filters:
    Month
    Day of week

-> Why Dashboard?
Dashboards help non-technical stakeholders:
Understand data quickly
Identify trends without reading tables
Make faster business decisions
